Event Overview

The SEAL Spring 2026 Hackathon is a high-intensity, 48-hour event designed to push the boundaries of student software engineering. Participants will form teams of up to four members to ideate, prototype, and present functional software solutions addressing real-world challenges across multiple tracks.

Eligibility & Requirements

  • Open to all currently enrolled undergraduate and graduate students.
  • Teams must consist of 2 to 4 members. Cross-disciplinary teams are highly encouraged.
  • All code must be written during the event duration. Pre-existing templates are allowed but must be declared.
  • Submissions must include a functional prototype and a public GitHub repository.
