Abstract

Our project proposes a localized cooling system for urban heat islands utilizing a network of low-cost IoT sensors and predictive machine learning models. By analyzing real-time temperature, humidity, and pedestrian traffic data, the system dynamically adjusts the output of localized evaporative cooling stations.

// Sensor Data Ingestion Pipeline (Sample)
function ingestSensorData(payload) {
  const normalizedData = {
    temp: payload.t * 1.8 + 32, // Convert to F
    humidity: payload.h,
    timestamp: new Date(payload.ts).toISOString(),
    node_id: payload.id
  };
  return Database.insert('sensor_metrics', normalizedData);
}
