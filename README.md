# commit SEAL theo cú pháp:

| Loại | Khi nào dùng |
|------|-------------|
| **Story** | Requirement từ góc nhìn user — "Tôi muốn làm X để đạt Y" |
| **Task** | Việc kỹ thuật cụ thể, thường là breakdown của Story |
| **Bug** | Hành vi sai so với mong đợi, cần fix |
| **Spike** | Chưa biết cách làm → cần research/PoC trước |
| **Chore** | Việc bảo trì hệ thống, không thêm tính năng mới |
